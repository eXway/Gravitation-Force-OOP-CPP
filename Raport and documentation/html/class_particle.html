Detailed Description

Class for Particle (or just element) with needed parameters.

Parameters
x= absolute (in px) horizontal position
y= absolute (in px) vertical position
r= radius of element (visual purpuses)
vx= velocity (or delta position) in horizontal vector
vy= velocity (or delta position) in vertical vector
m= true mass for calculating force
rest= rest parameters was made for relational positioning, not absolute
